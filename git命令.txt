git的一些常用命令



git init				//初始化

git status				//查看状态，如果显示的中文乱码，可以执行 git config --global core.quotepath false




1、使用git添加文件到github

git add abc.txt  或者  git add -A  （上传次目录下的所有文件），-A（all）

git commit -m "更新的信息"

git push



2、删除文件

git rm abc.txt

git commit -m '更新的信息'

git push





git log		                //提交成功后，查看日志






git clone SSH地址		//克隆项目
